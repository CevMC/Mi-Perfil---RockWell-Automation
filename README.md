# Mi-Perfil---RockWell-Automation
Aquí esta toda mi información personal después de finalizar el curso de Programación con Rockwell Automation

El curso me pareció una muy buena oportunidad para crecer tanto laboralmente como en mis conocimientos. Tuve grandes tutores y profesores con un amplio dominio de los temas, de quienes aprendí muchísimo.
En general, fue un curso muy bueno.

Me gustaría que no cambiaran tan seguido a los profesores y que el curso fuera un poco más intensivo. También sería útil que, desde el comienzo, evaluaran nuestros conocimientos para evitar repetir contenidos que algunos ya dominan.
